# لوحة تحكم المحتوى الذكية (Smart Content & Promo Dashboard)

ملخّص
----
هذا المشروع يهدف إلى تحويل الريبو إلى "لوحة تحكم وعرض" احترافية مدعومة بذكاء اصطناعي لإنتاج محتوى ترويجي (صور، فيديوهات قصيرة، عروض صور/فيديو مع تايمر، تصدير لمنصات التواصل، أدوات تحرير صور ذكية، مؤثرات صوتية/أصوات مشاهير).

الحالة الحالية
-------------
- تمّ إنشاء هذا README الأولي كبداية للمشروع. بعد ذلك سيُضاف فرع "feature/smart-dashboard-scaffold" يحتوي على سكفولد مبدئي للمشروع (واجهة Next.js، backend بسيط، نقاط رفع الملفات).

ميزة أصوات المشاهير
--------------------
بناءً على طلبك، تم تفعيل الخطة التقنية لميزة "أصوات المشاهير" ضمن مخطط المشروع. ملاحظة قانونية مهمة: تشغيل أو نشر أو استخدام أصوات تشبه مشاهير حقيقيين يتطلب دائماً موافقات وترخيص صريح من صاحب الحق (المالك أو الوارث أو الجهة المصرّح لها). تقنيًا سنوفر دعمًا فنيًا مباشرًا لدمج خدمات voice-cloning مثل ElevenLabs أو Azure Neural TTS، لكن لا يجوز تفعيل أي صوت في بيئة الإنتاج التجاريّة إلا بعد رفع مستندات الترخيص والموافقة عليها صراحة داخل لوحة تحكم الإدارة. بموافقتك الصريحة هنا أنت تقر بالمسؤولية القانونية لإدارة وتشغيل هذه الميزة.

التالي
------
1. سأقوم الآن بإنشاء فرع "feature/smart-dashboard-scaffold" ودفع سكفولد مبدئي يحتوي على الملفات الأساسية (ARCHITECTURE.md, .env.example, package.json, pages/index.jsx, server/index.js).
2. بعد ذلك سأفتح Pull Request للسكفولد حتى تتمكن من مراجعة الأكواد والتهيئة.

ملاحظات نهائية
--------------
- لتشغيل كامل الميزات ستحتاج إلى مفاتيح API لخدمات: Cloudinary/S3, FFmpeg (موضّب محلي أو حاوية docker), OpenAI/Replicate/Stability for image processing, ElevenLabs/Azure for TTS, وAPIs لمنصات التواصل لنشر المحتوى (YouTube Data API, Facebook Graph API, X API, TikTok API).
- إذا أردت أفعّل ميزة "أصوات المشاهير" فورًا في بيئة الاختبار (staging) مع فرض رفع مستند الترخيص، سأجهز واجهة رفع المستندات وميزة التفعيل الإداري.
